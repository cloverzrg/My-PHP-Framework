# My-PHP-Framework  
一个轻量的 MVC 框架

### 写这个框架的目的  
1. 加强自己的 PHP 技能
2. 研究其他 PHP 框架源码的时候更轻松，通过对比相信收获也会更多
3. 希望以后做小项目可以使用这个框架

### 参考  
1. 从零开始打造自己的PHP框架 [慕课网]  
2. thinkphp 5.0 源码  

### 完成的功能
1. Config 配置类
2. Request 请求类
3. CSRF 跨域防护类
4. Cache 缓存类
5. Route 路由类
6. Redis 单例连接


### 环境
1. php7
2. nginx
3. composer
4. redis
5. mysql
